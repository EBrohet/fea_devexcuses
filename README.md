Ceci est un projet de test pour ForEach Academy.
Il s'agit des "Excuses de dev", un générateur de phrases d'excuse"

Pour lancer le projet :

- lancer la commande 'npm start' depuis le dossier backend
- lancer la commande 'npm start' depuis le dossier frontend

Si ce n'est pas automatique, ouvrir la page 'http://localhost:3000' dans le navigateur.
